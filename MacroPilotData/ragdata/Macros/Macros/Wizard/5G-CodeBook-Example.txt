#Code Book Format: 
#------------------------------------------------------------
# ! beam-id 
# beam-id should only contain characters allowed in filenames, except brackets, parantheses or underscore characters.
# Port No. | Antenna Module | Excitation State | Amplitude (dBm) | Phase (Degrees) | Frequency (GHz) | Polarization (optional)
#
# For each beam, all ports of each module need to be listed, those not part of the beam set to "off".
# Polarizaion is optional and can be used for phase sweep
#------------------------------------------------------------
# + <tsp-id> <beam-id1> ... <beam-idN>   Total Scan Pattern, considering multiple beams 
#        first parameter behind the + is the label for the tsp to be generated, followed by list of beam-ids or keyword "all" to consider all beams
#------------------------------------------------------------
# The following modifiers can be used optionally before each beam definition:
#
# *t            Switches Phase column to time shift in nano seconds
# *p            Switches back Phase column to phase in degree
# +a <number>   Adds <number> dB to the following amplitudes below
#------------------------------------------------------------
+ tsp-1 beam-1-1 beam-1-2
+ tsp-2 beam-2-1 beam-2-2
+ tsp-all all
#
